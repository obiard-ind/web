# Exercices
## Installation et première application

1. Créez votre environnement virtuel Python
2. Activez cet environnement
3. Installez Flask
4. Créez votre première application Flask
5. Démarrez votre serveur web 
6. Testez-le dans votre navigateur
7. Arrêtez votre serveur
8. Désactivez (quittez) votre environnement virtuel.

## Des routes... pas en déroute !
1. Créez les routes correspondant aux chemins suivants :
	- '/'
	- '/equipe'
	- '/logiciels'
	- '/logiciels/opensource'
	- /hardware'
	- '/hardware/reseau'
	- '/hardware/ordinateur' 
2. Pour chaque route, créez la fonction associée qui va y répondre

| Route                  | Description de ce que retourne la fonction        |
| ---------------------- | ------------------------------------------------- |
| '/'                    | un petit mot de bienvenue sur votre site          |
| '/equipe'              | présentation de qui vous êtes                     |
| '/logiciel'            | présentation de la rubrique                       |
| '/logiciel/opensource' | une liste de logiciels opensource                 |
| '/hardware'            | présentation de la rubrique consacrée au hardware |
| '/hardware/ordinateur' | une liste du hardware constituant un ordinateur   |
| '/hardware/reseau'     | une liste de hardware réseau                      |
**Rem** : pour chaque entrée, quelques lignes de texte suffisent.  Celles-ci peuvent mixer du code html et des styles pour agrémenter la mise en forme.
**Rem** : pour les listes, utilisez les balises html `<ul></ul>` pour afficher celles-ci.
